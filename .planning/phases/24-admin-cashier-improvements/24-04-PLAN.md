Allow admin to void (reverse) a completed transaction.

When a transaction is voided:
1. The transaction row in the Transactions Log sheet has its TransactionType changed to 'Voided'
2. The student's balance in the Money Accounts sheet is restored by the transaction amount
3. The admin UI updates the row to reflect the voided state

Purpose: Cashier mistakes (wrong student, duplicate charge, incorrect amount) currently require manual Google Sheets editing to fix.
Output: `POST /api/transactions/<txn_id>/void` route + Void button in `transactions.html`.

**Transactions Log sheet columns (11 cols):**
`TransactionID | Timestamp | MoneyCardNumber | TransactionType | Amount | BalanceBefore | BalanceAfter | StudentID | CashierID | Note | ItemsJson`

Column indices (1-based):
- Col 1: TransactionID
- Col 4: TransactionType (e.g. 'Purchase', 'Top-Up', 'Manual')
- Col 5: Amount (negative for purchases, positive for top-ups)
- Col 6: BalanceBefore
- Col 7: BalanceAfter
- Col 8: StudentID (used to find student's money account for balance reversal)
- Col 3: MoneyCardNumber (RFID UID, used to find money account)

**Money Accounts sheet columns:** MoneyCardNumber | StudentID | Balance | ...

**Void logic:**
- Find transaction row by TransactionID (col 1)
- Check it is not already 'Voided'
- Read Amount (col 5) — this is the deduction (negative for purchase, positive for topup)
- Reversal: student balance += (-Amount), i.e. add back the deducted amount
- Update TransactionType (col 4) to 'Voided'
- Find student's Money Account row by MoneyCardNumber (col 3) or StudentID (col 8) and update Balance

**Finding the money account row:** Look up MoneyCardNumber in Money Accounts sheet col 1.

**ID format:** TransactionID is like `TXN-20260309-001`. It is stored in column 1.

**Authentication:** `@admin_only`.

```python
@app.route('/api/transactions/<txn_id>/void', methods=['POST'])
@admin_only
def void_transaction(txn_id):
    """Void a transaction: mark it Voided in Transactions Log and reverse the student balance."""
    try:
        db = get_sheets_client()
        txn_sheet = db.worksheet('Transactions Log')
        all_rows = txn_sheet.get_all_values()  # includes header
        
        if not all_rows:
            return jsonify({'success': False, 'error': 'No transactions found'}), 404
        
        headers = all_rows[0]
        # Locate column indices (0-based for list, +1 for Sheets API)
        def col_idx(name):
            return headers.index(name) if name in headers else None
        
        txn_id_col = col_idx('TransactionID')
        type_col = col_idx('TransactionType')
        amount_col = col_idx('Amount')
        card_col = col_idx('MoneyCardNumber')
        
        row_index = None
        txn_row = None
        for i, row in enumerate(all_rows[1:], start=2):  # start=2: skip header, 1-based
            if txn_id_col is not None and len(row) > txn_id_col and row[txn_id_col] == txn_id:
                row_index = i
                txn_row = row
                break
        
        if row_index is None:
            return jsonify({'success': False, 'error': f'Transaction {txn_id} not found'}), 404
        
        current_type = txn_row[type_col] if type_col is not None else ''
        if current_type == 'Voided':
            return jsonify({'success': False, 'error': 'Transaction is already voided'}), 409
        
        # Parse amount (may be stored as string with - sign)
        try:
            amount = float(txn_row[amount_col]) if amount_col is not None else 0
        except (ValueError, TypeError):
            amount = 0
        
        # Mark transaction as Voided
        txn_sheet.update_cell(row_index, (type_col or 3) + 1, 'Voided')
        
        # Reverse the balance: find Money Account by card number
        card_number = txn_row[card_col] if card_col is not None else ''
        if card_number:
            try:
                accounts_sheet = db.worksheet('Money Accounts')
                accounts = accounts_sheet.get_all_records()
                acc_headers = accounts_sheet.row_values(1)
                balance_col_idx = acc_headers.index('Balance') + 1 if 'Balance' in acc_headers else 3
                
                for j, acc in enumerate(accounts):
                    if str(acc.get('MoneyCardNumber', '')) == str(card_number):
                        acc_row = j + 2  # +1 header, +1 1-based
                        current_balance = float(acc.get('Balance', 0) or 0)
                        # Reverse: add back the deducted amount (amount is negative for purchases)
                        new_balance = round(current_balance + (-amount), 2)
                        accounts_sheet.update_cell(acc_row, balance_col_idx, new_balance)
                        break
            except Exception as bal_err:
                logger.warning(f'void_transaction: balance reversal failed for {txn_id}: {bal_err}')
                # Still return success — transaction is voided even if balance update failed
        
        return jsonify({
            'success': True,
            'message': f'Transaction {txn_id} voided',
            'reversed_amount': -amount
        })
    except Exception as e:
        logger.error(f'void_transaction error: {e}', exc_info=True)
        return jsonify({'success': False, 'error': 'Failed to void transaction'}), 500
```

```javascript
document.querySelector('#transactionsTable tbody')?.addEventListener('click', async (e) => {
  const btn = e.target.closest('.void-btn');
  if (!btn) return;
  const txnId = btn.dataset.txnId;
  if (!confirm(`Void transaction ${txnId}?\nThis will reverse the student's balance.`)) return;
  btn.disabled = true;
  btn.textContent = '...';
  try {
    const res = await fetch(`/api/transactions/${txnId}/void`, {method: 'POST'});
    const data = await res.json();
    if (data.success) {
      // Replace button with Voided badge
      btn.replaceWith(Object.assign(document.createElement('span'), {
        className: 'badge bg-secondary',
        textContent: 'Voided'
      }));
      // Update the TransactionType cell in the same row
      const row = btn.closest ? btn.closest('tr') : null;
      if (row) {
        const cells = row.querySelectorAll('td');
        // Find and update the type cell (typically 3rd or 4th column)
        cells.forEach(cell => {
          if (['Purchase', 'Manual', 'NFC Purchase'].includes(cell.textContent.trim())) {
            cell.textContent = 'Voided';
          }
        });
      }
    } else {
      alert(data.error || 'Failed to void transaction');
      btn.disabled = false;
      btn.textContent = 'Void';
    }
  } catch {
    alert('Network error');
    btn.disabled = false;
    btn.textContent = 'Void';
  }
});
```
