**Core/Network/APIClient.swift:**
- Class conforming to ObservableObject (so it can be injected via @EnvironmentObject)
- Private `URLSession.shared`
- Private helper `authenticatedRequest(path:method:body:)` that reads `KeychainHelper.read(forKey: "auth_token")` and sets `Authorization: Bearer <token>` header
- Private helper `perform<T: Decodable>(_ request: URLRequest) async throws -> T` that: executes the request, checks for 403 → decodes body as `[String:String]` → if `body["error"] == "CARD_LOST"` throws `APIError.cardLost`, else throws `APIError.unauthorized`; checks for other non-2xx → throws `APIError.httpError(statusCode)`; decodes response as T
- Public methods (all `async throws`): `login(studentId:pin:)`, `getBalance()`, `getTransactions(limit:offset:)`, `getBudget()`, `setBudget(monthlyLimit:)`, `reportLostCard()`, `logout()`
- CRITICAL: Never use `₱` — use `฿` if any format string is needed here
