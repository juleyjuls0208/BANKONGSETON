**In `nfc_pay()` — replace the bare `nfc_service.get_virtual_card_by_token()` call (line ~1056) with a cache-first lookup:**

```python
# Step 2: Look up VirtualCard by token (must be active)
db = get_sheets_client()
_vc_records = get_cached("virtual_cards_all")
if _vc_records is None:
    _vc_records = get_worksheet_with_retry("VirtualCards").get_all_records()
    set_cached("virtual_cards_all", _vc_records, ttl=30)
matched = next(
    (
        r for r in _vc_records
        if r.get("VirtualCardToken") == virtual_card_token
        and str(r.get("IsActive", "")).upper() == "TRUE"
    ),
    None,
)
if not matched:
    return jsonify({"error": "Invalid or inactive virtual card token"}), 401
```

This replaces `db = get_sheets_client()` + `matched = nfc_service.get_virtual_card_by_token(...)`.

**Eliminate the redundant Money Accounts header call:**
Currently `nfc_pay()` calls `money_sheet.get_all_records()` (line ~1068) to get records, then later calls `money_sheet.row_values(1)` (line ~1093) just to find the "Balance" column index. Eliminate `row_values(1)` entirely — derive the column index from the `records` dict keys instead:

Replace:
```python
header_row = money_sheet.row_values(1)
try:
    balance_col_idx = header_row.index("Balance") + 1
except ValueError:
    balance_col_idx = 3  # fallback: column C
```
With:
```python
# Derive column index from record keys (avoids extra API call)
if money_records:
    header_keys = list(money_records[0].keys())
    try:
        balance_col_idx = header_keys.index("Balance") + 2  # +2: gspread rows are 1-based and data starts at row 2
    except ValueError:
        balance_col_idx = 3  # fallback: column C
else:
    balance_col_idx = 3
```

Wait — gspread `get_all_records()` returns a list of dicts where keys are the column headers. The column index in the sheet = position in the first row. The key order from `get_all_records()` dicts matches header order in Python 3.7+. Use `list(money_records[0].keys()).index("Balance") + 1` for the 1-based column index (no +2 needed; column index is just 1-based header position, unrelated to row number).

Correct replacement:
```python
# Derive Balance column index from record keys — avoids extra row_values(1) API call
if money_records:
    try:
        balance_col_idx = list(money_records[0].keys()).index("Balance") + 1
    except ValueError:
        balance_col_idx = 3
else:
    balance_col_idx = 3
```
