Add the complete JavaScript block before `</body>`. Implement these functions:

**showToast(message, type='success'):**
```javascript
function showToast(message, type = 'success') {
    const toastEl = document.getElementById('saveToast');
    document.getElementById('toastMessage').textContent = message;
    toastEl.className = `toast align-items-center text-bg-${type} border-0`;
    new bootstrap.Toast(toastEl, { delay: 2000, autohide: true }).show();
}
```

**loadProducts():** fetch GET `/api/products/list`, populate `#productsTableBody` using `renderTable(products)`.

**renderTable(products):** Build table rows dynamically. Each row:
```html
<tr data-id="${p.id}" class="${p.active ? '' : 'table-secondary'}">
  <td class="editable-cell" data-field="name" data-id="${p.id}">
    <span class="cell-display">${p.name}</span>
    <input class="cell-input form-control form-control-sm" value="${p.name}"
           onblur="saveField(this)" onkeydown="if(event.key==='Enter')this.blur()">
  </td>
  <td class="editable-cell" data-field="category" data-id="${p.id}">
    <span class="cell-display">${p.category}</span>
    <select class="cell-input form-select form-select-sm"
            onblur="saveField(this)" onchange="saveField(this)">
      ${['Food','Drinks','Snacks','Other'].map(c =>
        `<option${c===p.category?' selected':''}>${c}</option>`).join('')}
    </select>
  </td>
  <td class="editable-cell" data-field="price" data-id="${p.id}">
    <span class="cell-display">₱${p.price.toFixed(2)}</span>
    <input type="number" step="0.01" class="cell-input form-control form-control-sm"
           value="${p.price}"
           onblur="saveField(this)" onkeydown="if(event.key==='Enter')this.blur()">
  </td>
  <td>
    <span class="badge status-badge ${p.active ? 'bg-success' : 'bg-secondary'}">
      ${p.active ? 'Active' : 'Inactive'}
    </span>
  </td>
  <td>
    <div class="form-check form-switch mb-0">
      <input class="form-check-input" type="checkbox" role="switch"
             ${p.active ? 'checked' : ''}
             onchange="toggleStatus('${p.id}', this.checked)">
    </div>
  </td>
</tr>
```

Add click handler on `<td class="editable-cell">` to toggle editing class:
```javascript
document.getElementById('productsTableBody').addEventListener('click', function(e) {
    const td = e.target.closest('.editable-cell');
    if (!td || td.classList.contains('editing')) return;
    td.classList.add('editing');
    const input = td.querySelector('.cell-input');
    input.focus();
    if (input.select) input.select();
});
```

**saveField(input):** On blur, POST to `/api/products/update` with `{id, [field]: value}`. On success: update `.cell-display` text, show "Saved" toast, remove `editing` class. On failure: show "Save failed" toast (type='danger'), restore original value, remove `editing` class.

Debounce guard: add a `saving` Set at module level; skip call if `saving.has(productId)`; add before fetch, delete after fetch.

**addProduct(event):** Prevent default. Validate Name and Price non-empty (add `is-invalid` class; remove on any input event). Generate `id = 'PROD-' + Date.now()`. POST to `/api/products/update` with `{id, name, category, price: parseFloat(price), active: true}`. On success: call `loadProducts()` to refresh table, reset form, show "Product added" toast. On failure: show error toast.

**toggleStatus(productId, active):** POST to `/api/products/toggle-status` with `{id: productId, active}`. On success: toggle `table-secondary` class on the row, update status badge text and class, show "Product activated"/"Product deactivated" toast. On failure: revert the checkbox state, show error toast.
