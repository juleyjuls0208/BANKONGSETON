   `DiffUtil.calculateDiff()` run on `Dispatchers.Default`, dispatching results to main thread.

   `{"monthly_spend": float}`. Update `updateBudgetUI()` signature accordingly.

Purpose: DiffUtil reduces RecyclerView redraws to only changed rows. The budget summary
endpoint eliminates the 200-transaction payload just to compute one number.

```kotlin
private class TransactionDiffCallback(
    private val oldList: List<Transaction>,
    private val newList: List<Transaction>
) : DiffUtil.Callback() {
    override fun getOldListSize() = oldList.size
    override fun getNewListSize() = newList.size

    override fun areItemsTheSame(oldPos: Int, newPos: Int): Boolean {
        val old = oldList[oldPos]
        val new = newList[newPos]
        // Composite identity key: timestamp + type + amount
        return old.timestamp == new.timestamp &&
               old.type == new.type &&
               old.amount == new.amount
    }

    override fun areContentsTheSame(oldPos: Int, newPos: Int): Boolean {
        // Transaction is a data class — == does structural equality
        return oldList[oldPos] == newList[newPos]
    }
}
```

**Replace `setTransactions()` (delete the old 3-line implementation):**
```kotlin
fun setTransactions(newTransactions: List<Transaction>) {
    val oldList = transactions.toList()  // snapshot for DiffUtil
    adapterScope.launch {
        val diffResult = withContext(Dispatchers.Default) {
            DiffUtil.calculateDiff(TransactionDiffCallback(oldList, newTransactions))
        }
        // Back on main thread
        transactions.clear()
        transactions.addAll(newTransactions)
        diffResult.dispatchUpdatesTo(this@TransactionsAdapter)
    }
}
```
