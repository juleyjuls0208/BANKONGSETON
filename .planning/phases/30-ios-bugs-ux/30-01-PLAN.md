    // It is called from the View's alert button action so the alert

```swift
// Before:
private func clearAll() {
// After:
func clearAll() {

`clearAll()` is only called from `logout()` internally and will also be called from the new alert button in Plan 02. No existing callers are affected.

**CRITICAL:** Do NOT call `clearAll()` inside `handleUnauthorized()`. That ordering would set `isLoggedIn = false` before the user sees the alert, destroying the view.

    if let cached = KeychainHelper.read(forKey: "last_balance"),
       let value = Double(cached) {
        balance = value
    }
}
```
This initialises `balance` from the Keychain key `last_balance` (already written on every successful load) so the UI shows the last-known balance instead of ₱0.00 during the first load.

B) In `load()`, add a `.unauthorized` catch BEFORE the generic `catch` block:
```swift
} catch APIError.unauthorized {
    authManager.handleUnauthorized()
```
