## Decisions Made
- Wrapped serial import in `try/except ImportError` rather than removing it — allows `web_app.py` to also function locally with hardware if pyserial is available
- Kept Flask-SocketIO in `web_app.py` — WebSocket upgrade requests on PythonAnywhere are silently rejected (acceptable behavior, no crash)
- Removed `socketio.emit()` calls only from HTTP route handlers, not from `@socketio.on` handlers — preserves real-time UI for local/desktop use
- Added `'bangko-jwt-secret-2026'` as the insecure-default sentinel in the JWT_SECRET guard (per CONTEXT.md locked decision)
